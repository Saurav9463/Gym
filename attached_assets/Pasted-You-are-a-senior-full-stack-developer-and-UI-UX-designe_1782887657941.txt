## 🗄️ SUPABASE SCHEMA

Run this SQL to create all tables:

```sql
create table memberships (
  id uuid primary key default gen_random_uuid(),
  name text not null,
  duration text,
  price numeric,
  features text[],
  popular boolean default false,
  active boolean default true
);

create table trainers (
  id uuid primary key default gen_random_uuid(),
  name text not null,
  role text,
  bio text,
  speciality text,
  certifications text,
  photo_url text,
  years_experience int,
  active boolean default true
);

create table bookings (
  id uuid primary key default gen_random_uuid(),
  client_name text,
  client_email text,
  client_phone text,
  plan_id uuid references memberships(id) on delete set null,
  trainer_id uuid references trainers(id) on delete set null,
  appointment_date date,
  appointment_time time,
  status text default 'Pending',
  notes text,
  created_at timestamptz default now()
);

create table classes (
  id uuid primary key default gen_random_uuid(),
  name text,
  trainer_id uuid references trainers(id) on delete set null,
  day_of_week text,
  start_time time,
  duration_minutes int,
  difficulty text,
  active boolean default true
);

create table gallery (
  id uuid primary key default gen_random_uuid(),
  image_url text,
  category text,
  created_at timestamptz default now()
);

create table messages (
  id uuid primary key default gen_random_uuid(),
  name text,
  email text,
  message text,
  read boolean default false,
  created_at timestamptz default now()
);

create table settings (
  key text primary key,
  value text
);
```

## 🔌 DATA FETCHING (CRITICAL)

DO NOT use any workspace API client, Express routes, or custom 
backend hooks. Use ONLY direct Supabase queries throughout the app.

Pattern for every data fetch:
  const [data, setData] = useState<any[]>([]);
  useEffect(() => {
    supabase.from("table_name").select("*")
      .then(({ data, error }) => {
        if (error) console.error(error);
        setData(data ?? []);
      });
  }, []);

For admin pages that need all records (not just active ones):
  DO NOT add .eq("active", true) filter — admin sees everything

For public pages: 
  ADD .eq("active", true) filter

After any insert/update/delete: immediately re-fetch 
the table to refresh the UI without page reload
